Explore Inkscape's Features

Discover the powerful tools Inkscape offers to boost your creativity

Create and Design: Draw with pencil and pen tools, and create shapes like rectangles, ellipses, stars, and spirals. Transform and Arrange: Manipulate objects with ease using transformation tools, layers, and alignment commands. Path Operations: Edit paths with node editing, path simplification, and apply Live Path Effects. Text and Typography: Use advanced text tools for multi-line text and on-canvas editing. Colors and Styles: Access various color selectors, gradient editors, and pattern fills to style your designs. File Compatibility: Import and export multiple file formats like SVG, PNG, and PDF.

Our Promise

At Inkscape, we are dedicated to transparency, privacy, and accessibility. As Free Open Source Software (FOSS), our code is open for everyone to view and contribute to, ensuring complete transparency. We respect your privacy by guaranteeing no telemetry or tracking within our software, so your data remains your own. Best of all, Inkscape is free forever, providing all its powerful features at no cost to everyone.
